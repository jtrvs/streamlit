# Интерактивный анализатор аминокислотного состава белка
1. Принимает на вход белковую последовательность в формате FASTA (через текстовое поле или загрузку файла).
2. Вычисляет частоту каждой аминокислоты.
3. Считает основные свойства: молекулярную массу (приближенно), заряд при pH 7, гидрофобность.
4. Строит интерактивные графики (гистограмма состава, круговая диаграмма классов аминокислот).
5. Сохраняет результат анализа в виде CSV (опционально).

## Установка окружения
conda create -n streamlit python=3.12 streamlit pandas matplotlib plotly biopython

## Запуск приложения
streamlit run protein_analyzer.py
